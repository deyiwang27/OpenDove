# Vision

## Mission

OpenDove exists to help humans and AI build the right software with minimal manual coordination.

It is designed for an AI-native workflow where coding assistants are already effective at local execution, but software delivery as a whole still requires too much human supervision to stay aligned with user intent.

## Long-Term Vision

OpenDove aims to become an autonomous software factory.

In that future state, users provide goals, constraints, and policy boundaries. OpenDove manages the software delivery loop from specification through validation, involving humans only when intent is unclear, risk is high, or explicit approval is required.

## Near-Term Starting Point

OpenDove starts as a developer tool for semi-autonomous engineering teams.

The initial goal is not to replace humans completely. The initial goal is to remove the manual coordination burden around AI-assisted delivery while preserving trust, traceability, and alignment.

## Core Thesis

Current coding agents are strong builders but weak delivery systems.

They can write code, run tools, and solve local implementation problems. What is still missing is a system that can:

- preserve the original goal across multiple stages
- assign responsibility clearly
- decide when work is actually complete
- reject output that is plausible but misaligned
- escalate when ambiguity cannot be resolved safely

OpenDove is intended to provide that missing layer.

## Product Belief

The future of software development will not be one agent replacing the whole process.

It will be a managed system where specialized AI capabilities are orchestrated inside a stronger control loop. In that loop, execution becomes cheaper and faster, but alignment, validation, and escalation become more important than ever.

## Strategic Direction

OpenDove should position itself as the orchestration and alignment system around AI coding assistants such as Claude Code, Codex, and Gemini.

It should not compete by trying to be a better chat interface or a slightly different coding copilot. Its distinct value is managing the delivery loop, not only generating code.

## One-Sentence Vision

OpenDove turns powerful coding assistants into a trustworthy software delivery system.
