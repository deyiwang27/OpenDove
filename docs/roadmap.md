# Roadmap

## v0

- Define the task schema and state machine
- Persist tasks and events in PostgreSQL
- Run one orchestrated flow through all five roles
- Enforce AVA approval as a blocking gate

## v1

- Add real tool execution for implementation agents
- Add retry and escalation handling
- Add richer integration validation

## v2

- Support multiple concurrent tasks
- Add dashboards for state and trace inspection
- Add more granular agent specializations if needed
